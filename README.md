# Menyoo FiveM
### Fork from MenyooSP
#### Trainer for FiveM
---

## Requirements
- [FiveM](https://fivem.net/)

## Download / Install
- Grab the [latest release](https://github.com/PichotM/MenyooFiveM/releases).
- Place Menyoo.asi in the plugins folder of FiveM.
- Place menyooStuff in data/cache/subprocess folder of FiveM.
